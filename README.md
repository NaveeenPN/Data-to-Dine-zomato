This project presents an end-to-end Power BI dashboard built using Zomato data to analyze sales performance, user behavior, restaurant demand, and city-wise trends. The dashboard provides interactive insights that help understand how different factors such as city, age group, food category, and time impact business performance.

### Key Objectives
- Analyze overall sales, orders, ratings, and quantity trends
- Understand user acquisition and churn patterns
- Identify top-performing cities and restaurants
- Study customer behavior by age group and food preference

### Key KPIs
- Total Sales: 987M  
- Total Orders: 150K  
- Total Quantity: 2M  
- Total Ratings: 148K  
- Active Users: 78K  

### Dashboard Highlights
- **Sales Analysis:** Year-wise, date-wise, and age-wise sales trends
- **City Performance:** Top cities by sales, orders, ratings, gained and lost users
- **User Analysis:** Active users, gained vs lost users, gender and age distribution
- **Category Insights:** Veg, Non-Veg, and Other food category performance
- **Comparative Analysis:** Previous vs current year sales comparison
- **Geographic View:** Map-based visualization for gained and lost users
- **Interactive Features:** Slicers, drill-downs, top-N filters, and Q&A visual

### Tools & Technologies
- Power BI
- SQL
- Excel / CSV

### Business Insights
- Identified cities contributing the highest revenue and user growth
- Highlighted peak age groups driving maximum sales
- Analyzed user churn to understand retention challenges
- Compared yearly performance to track business growth and decline
